RetroBlaster Firmware Changelog
----------------------------
----------------------------
Version 2.8d
----------------------------
- Fixed a bug where GBC games larger than 4MByte would fail to write properly.

----------------------------
Version 2.8c
----------------------------
- Added NES MAKER (mapper 30 / UNROM-512) support! You can now read/write any NES Maker-compatible hardware using the RetroBlaster 2.0.

----------------------------
Version 2.8a & 2.8b
----------------------------
- bugfixes

----------------------------
Version 2.8
----------------------------
- Added GBA EEPROM/Flash support
- NES MMC6 support
- NES MMC Blaster Lite and NES Discrete Blaster Pro support

- Turbo Blaster Flash ID fix - will not be able to write on older firmware!

- Fix Cybernoid error, now flashes properly to NES Blaster Lite CNROM boards

----------------------------
Version 2.7
----------------------------
- Fixed an intermittent N64 Blaster issue where the CIC type wasn't being changed when writing a game.

----------------------------
Version 2.6
----------------------------
- Improved reliability of N64 EEPROM dumping
- Added N64 GameShark dumping/writing
- PCE/TG16 finalized for upcoming Turbo Blaster and HuCard adapter.

----------------------------
Version 2.5
----------------------------
- Fixed issue with N64 FlashRAM chips not dumping correctly.

----------------------------
Version 2.4
----------------------------
NEW:
- Added Turbo Grafx 16 / PC Engine support
- Sega Genesis SRAM writing support

IMPROVEMENTS:
- 25% speed increase in writing games to SNES Blaster
- 15% speed increase in writing games to GB Blaster

FIXES:
- fixed NES MMC4 header mismatch
- fixed bug that wouldn't allow writing Sega games with odd ROM headers
- added ".md" ROM extensions as one of the defaults for Sega Blaster

- fixed annoying bug that was affecting SNES writing and dumping. Games
like Super Metroid and Yoshi's Island (among others) now work

----------------------------
Version 2.3a
----------------------------
- New naming system to avoid confusion between software and firmware releases.
- Added SNES cart ROM dumping support (doesn't support enhancement chips)
- Added SNES cart SRAM read/writing support
- NES MMC Blaster support!

----------------------------
Version 2.0.2	  
----------------------------
- Added new NES mapper support.
- Stability fixes.
- GB/GBC dump fixes and save support.


----------------------------
Version 2.0.1
----------------------------
- Added support for NES,GB/GBC,GBA,SNES,SEGA GENESIS. 


----------------------------
Version 2.0c-beta
----------------------------
- workaround for instability on some computers, only use if version 2.0b doesn't work


----------------------------
Version 2.0b
----------------------------
- initial firmware release
