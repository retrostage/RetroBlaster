RetroBlaster Firmware Changelog
----------------------------

----------------------------
Version 2.6
----------------------------
- Improved reliability of N64 EEPROM dumping
- Added N64 GameShark dumping/writing
- PCE/TG16 finalized for upcoming Turbo Blaster and HuCard adapter.

----------------------------
Version 2.5
----------------------------
- Fixed N64 FlashRAM dumping problem, all games should now dump correctly.

----------------------------
Version 2.4
----------------------------
NEW:
- Added Turbo Grafx 16 / PC Engine support
- Sega Genesis SRAM writing support

IMPROVEMENTS:
- 25% speed increase in writing games to SNES Blaster
- 15% speed increase in writing games to GB Blaster

----------------------------
Version 2.3a
----------------------------
- New naming system to avoid confusion between software and firmware releases.
- Added SNES cart ROM dumping support (doesn't support enhancement chips)
- Added SNES cart SRAM read/writing support
- NES MMC Blaster support!

----------------------------
Version 2.0.2	  
----------------------------
- Added new NES mapper support.
- Stability fixes.
- GB/GBC dump fixes and save support.


----------------------------
Version 2.0.1
----------------------------
- Added support for NES,GB/GBC,GBA,SNES,SEGA GENESIS.
